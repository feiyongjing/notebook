~~~C++
// 内联函数和普通函数一样，但是会在适当的时机向宏一样展开，没有普通函数的调用的开销，也不会向宏函数一样进行替换导致语法错误，

// 函数的声明和实现必须都添加inline关键字修饰，那么这个函数可能会被当成内联函数（具体由编译器决定，但是内联函数有以下特点）
// 1、函数体不能存在任何形式的循环语句
// 2、函数体不能出现过多的条件判断语句
// 3、函数体不能过长
// 4、不能对函数进行取地址操作
// 注意默认类中的函数隐式添加了inline关键字修饰，但是不一定的内联函数
inline int fun1();
inline int fun1() {
	return 100;
}
~~~
