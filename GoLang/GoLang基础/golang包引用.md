## 包和包引用
- 一个文件后缀是 .go 的文件必须声明该文件的包

### 自定义的包
~~~go
package p1

import "fmt"

// init函数是包的初始化函数，每个包被引用时被优先调用
// 因此传递性依赖会根据依赖链路最先执行最底层包的init函数，如果两个包的依赖链路层级一样会根据包依赖的引用顺序先后执行包的init函数
func init() {
	fmt.Println("init p1")
}

// 函数的首字母表示访问控制范围，大写表示公有的函数，小写表示包私有函数
// 注意在同一个包下不能有函数名相同的函数以及相同的名称的全局变量名称，这会报重复定义的错误
// go中是不支持传统的函数重载，即相同的函数名，而参数列表的数量和类型不同
func Test1() {
	fmt.Println("test1")
}
~~~

### 包应用
~~~go
// 程序启动的入口, 必须是main包，包的名称和文件以及目录没有关系，但是一般情况下包名应该需要和包目录名称保持一致
package main

// 注意引入包时，如果是自定义的包，需要写自定义包目录的路径（从目录项目目录名称开始或者是src目录开始），包别名可以不写同样可以直接使用包名调用内部的公有函数
// 如果包没有被使用则无法引用该包或者给该包的别名是 _
import (
	p1 "demo1/package1"
	"fmt"   // 字符串格式化包，一些系统包通常是在 GOPATH下的src目录下开始查找
)

// 程序入口
func main() {
	// 定义变量，如果变量定义了没有被使用则会报错
	// 海象运算符是用于变量类型推断，但是只能用于局部变量
	s := "哈哈"

	// Sprintf 根据格式化参数生成格式化的字符串并返回该字符串。
	// Printf 根据格式化参数生成格式化的字符串并写入标准输出。
	fmt.Println(s)
	p1.Test1()
}

func init() {
	fmt.Println("init main")
}
~~~
