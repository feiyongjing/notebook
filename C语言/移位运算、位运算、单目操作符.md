### 位运算
~~~C
& // 按位与 都是1的位就是1，否则是0

| // 按位或 只要有0的位就是1，否则是1

^ // 按位异或  相同的位是0，不同的位是1
// 由于 3^3=0 所有可以使用异或交换两个数
int a=3,b=5;
a=a^b;
b=a^b;
b=a^b;


~~~
