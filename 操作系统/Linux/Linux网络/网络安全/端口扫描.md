~~~shell
# 常见的端口扫描工具 nmap ，安装命令 apt install nmap 或 yum install namp -y
# 注意事项
#     在进行网络扫描时，确保你有适当的授权。未经授权的端口扫描可能会被视为恶意行为。
#     扫描可能会被目标系统的防火墙或入侵检测系统检测到，因此在敏感环境中操作时要格外小心。
#     Nmap功能强大，但也有一定的复杂性。查阅Nmap官方文档了解更多选项和用法。
# nmap 端口扫描，参考：https://blog.csdn.net/Xxy605/article/details/107620999
nmap [-Pn] [服务器IP或域名或网段] [-p 80,443 或 1-40000] [-sV]
# -sS 参数: 执行TCP SYN扫描，这是Nmap默认的扫描方式，通常被称为半开放扫描。
# -p 参数指定扫描端口或者是端口扫描范围
# -sV 参数探测主机的开放端口以确定该端口的服务和版本信息
# -sP 参数向指定主机或网段发送 ping 的ICMP数据包，确定那些主机存活
# -O 参数尝试识别目标主机的操作系统，需要root权限才能启用
# -A 参数: 启用操作系统检测、版本检测、脚本扫描和跟踪路由，需要root权限才能启用
# -Pn 参数: 跳过主机发现，直接进行端口扫描。
# -sN 参数：TCP空扫描欺骗防火墙
# -v参数: 提高输出信息的详细程度（可以使用多个v来增加详细程度）


### 默认端口一般用于那些服务
# 查看/etc/services系统文件，它映射了网络服务到它们对应的端口号和协议类型。通常被用来确定一个特定的端口号应该用于什么网络服务
cat /etc/services
~~~
