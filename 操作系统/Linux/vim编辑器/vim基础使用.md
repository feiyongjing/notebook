### Vim是一个功能强大的全屏幕文本编辑器，是Linux/UNIX上最常用的文本编辑器，它的作用是建立、编辑、显示文本文件。
### Vim没有菜单，只有命令。

### vim的中文文档查看手册命令 vimtutor

~~~shell
# 进入查看编辑文件内容
vim [文件名]

# vim有3种模式：命令模式、输入模式、底线命令模式。三种模式之间的切换如下：
# 执行命令一开始进入的页面是命令模式，在命令模式下按i、o、a进入输入模式，在命令模式下输入 : 进入底线命令模式
# 而在输入模式和底线命令模式下按键盘左上角esc键会进入命令模式
# 在底线命令模式下按wq加回车会保存文件修改并退出，按q加回车则不保存退出，按q!加回车则不保存强制退出，注意如果vim编辑的是一个不存在的文件，保存退出会创建这个文件而不保存退出则不会创建这个文件

# 命令模式操作
# 使用上下左右键或者是hjkl键（h向左、j向下、k向上、l向右）移动光标到下一个字符
# 使用w键移动光标到下一个单词字符的开始处
# 按a光标移动到所在字符之后并且进入插入模式
# 按A光标移动到所在行末尾并且进入插入模式
# 按i光标移动到所在字符之前并且进入插入模式
# 按I光标移动到所在行首并且进入插入模式
# 按o光标移动到所在行下插入新行并且进入插入模式
# 按O光标移动到所在行上插入新行并且进入插入模式
# 按gg进入文件的第一行
# 按G进入文件的最后一行
# 按nG进入文件的第n行
# 按0光标移动到行首
# 按$光标移动到行尾
# 按Ctrl+u向上翻半页
# 按Ctrl+d向下翻半页
# 按Ctrl+b向上翻一页
# 按Ctrl+f向下翻一页
# 按x删除光标所在处的字符
# 按nx删除光标所在处之后n个字符
# 按yy复制光标所在行
# 按nyy复制包括光标所在行之下n行
# 按dd剪切光标所在行
# 按ndd剪切包括光标所在行之下n行
# 按小写p粘贴在光标所在行的下一行
# 按大写P粘贴在光标所在行的上一行
# 按dG删除光标所在行到文件末尾
# 按D删除光标所在处到文件末尾
# 按r进入单个字符替换模式，再输入的第二个字符会替换光标所在字符。如果改错了需要重新按r进入替换模式
# 按R进入替换模式，接下来的输入会直接替换光标所在处的字符，如果修改好了可以按ecs键退出替换模式
# 按u回退到上一步，可以连续回退
# 按Ctrl+r撤销上一次的回退，可以连续撤销回退
# 输入/local会搜索字符串local，按n代表光标移动到下一个搜索字符串，按N代表光标移动到上一个搜索字符串，忽略大小写需要在先在底线命令模式下设置 set ic 回车，明确大小写需要在先在底线命令模式下设置 set noic回车
# 输入?local会搜索字符串local，按N代表光标移动到下一个搜索字符串，按n代表光标移动到上一个搜索字符串，忽略大小写需要在先在底线命令模式下设置 set ic 回车，明确大小写需要在先在底线命令模式下设置 set noic回车
# 输入#会搜索当前光标所在的字符串，按N代表光标移动到下一个搜索字符串，按n代表光标移动到上一个搜索字符串，忽略大小写需要在先在底线命令模式下设置 set ic 回车，明确大小写需要在先在底线命令模式下设置 set noic回车
# 格式化文件中的全部代码，输入gg光标移动到文件开头，按下=  再按下G 表示一直到文件末尾都格式化 
# 格式化文件中部分代码，按v进入可视模式，使用上下箭头选择部分代码，按下 = 格式化选中的代码
# 选择文本块，按v进入行可视模式（常见的单行开始光标），按Ctrl+v进入列可视模式（光标在使用上下键时会生成多行光标），在可视模式下可以使用选择代码块进行复制（可视模式下复制是y）和删除操作(可视模式下d是删除)
# 注释多行，先将光标移动到行首字符，使用Ctrl+v进入列可视模式选择需要注释的行，按大写的I进入插入模式输入注释的#或者是// , 再按两次ESC键就完成了多行注释
# 去除多行注释，先将光标移动到行首字符，使用Ctrl+v进入列可视模式选择需要去除注释的行注释字符 , 再按小写 d 就完成了去除多行注释
# 保存文件修改并退出按ZZ

# 底线命令模式操作
# 显示行号，在底线命令模式下输入 set number 加回车或者简写 set nu 加回车
# 取消行号，在底线命令模式下输入 set nonumber 加回车或者简写 set nonu 加回车
# 移动光标到第n行，在底线命令模式下输入 n行号加回车
# 删除n1行到n2行包括n1行和n2行，在底线命令模式下输入 n1,n2d 加回车
# 全文替换指定字符串，在底线命令模式下输入 %s/old/new/g加回车，old是原来的字符串new是新的字符串，最后的g换成c就会一个个询问是否要替换
# 全文替换指定字符串，在底线命令模式下输入 1,$s/old/new/g加回车，old是原来的字符串new是新的字符串，最后的g换成c就会一个个询问是否要替换
# 光标当前行替换第一个匹配的字符串，在底线命令模式下输入 s/old/new 加回车，old是原来的字符串new是新的字符串
# 光标当前行替换匹配的全部字符串，在底线命令模式下输入 s/old/new/g 加回车，old是原来的字符串new是新的字符串
# n1行到n2行范围替换指定字符串，在底线命令模式下输入 n1,n2s/old/new/g加回车，old是原来的字符串new是新的字符串最后的g换成c就会一个个询问是否要替换
# 保存文件修改，在底线命令模式下输入w 回车
# 文件修改另存为指定文件，在底线命令模式下输入w new_filename 回车
# 保存文件修改并退出，在底线命令模式下输入wq 回车 或者是 x回车
# 不保存文件修改并退出，在底线命令模式下输入q! 回车，!代表强制退出
# 保存文件修改并强制退出（只有root权限可以使用），在底线命令模式下输入wq! 回车
# 设置unix文件格式，在底线命令模式下输入set ff=unix
# 执行shell命令，在底线命令模式下输入 !shell命令 会在当初执行vim的目录下执行这条shell命令，并且显示结果，而继续按回车会回退到vim中
# 将其他的文件内容导入到光标之后，在底线命令模式下输入r /root/test.text 回车，/root/test.text文件的内容就导入到当前文件中光标之后的位置了
# 执行命令并得到命令执行结果，在底线命令模式下输入r ls -l 回车，会显示在vim 执行命令的路径下执行命令ls -l的结果
# 将命令的执行结果导入到光标之后，在底线命令模式下输入r !date 回车，会将date命令的执行结果导入到当前文件中光标之后的位置了
# 定义快捷输入，在底线命令模式下输入ab mymail feiyongjing@cloudgrow.cn 回车，会在输入模式下输入mymail回车后将mymail直接替换为feiyongjing@cloudgrow.cn，但是它并不是永久有效的，只要退出vim就失效了如果需要永久生效需要创建设置vim的配置，vim的配置是 ~/.vimrc文件

# 设置多行#注释，在底线命令模式下输入n1,n2s/^/#/g 回车
# 去除多行#注释，在底线命令模式下输入n1,n2s/^#//g 回车
# 添加多行//注释，在底线命令模式下输入n1,n2s/^/\/\//g 回车
# 去除多行//注释，在底线命令模式下输入n1,n2s/^\/\///g 回车
~~~
