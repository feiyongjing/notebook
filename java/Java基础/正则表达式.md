# 元字符：带有特殊意义的符号
~~~

.    匹配一个除换号以外的任意字符
\w   匹配一个字母或数字或下划线
\W   匹配一个非字母或非数字或非下划线
\s   匹配一个任意的空白字符
\S   匹配一个非空白字符
\d   匹配一个数字字符
\D   匹配一个非数字字符
\n   匹配一个换号符
\t   匹配一个制表符

^    匹配字符串的开始
&    匹配字符串的结束

a|b  匹配a字符或者是b字符

()      匹配括号中的表达式，也表示一个组
[...]   匹配字符组中的任意一个字符，当然也可以使用范围匹配，例如表示数字和字母使用 [a-zA-Z0-9]
[^...]  匹配字符组中的全部字符之外的一个字符

~~~
---

# 量词：控制前面的元字符出现的次数
~~~

*         重复0次到多次，例如表示0或者多个数字  \d*
+         重复1次到多次，例如表示1或者多个数字  \d+
?         重复0次或1次，例如表示0或者1个数字  \d?
{n}       重复n次，例如表示5个数字  \d{5}
{n,}      重复n次或更多次，例如表示5或更多个数字  \d{5,}
{n,m}     重复n次到m次，例如表示5到10个范围个数的数字  \d{5，10}


# 元字符后只有一个量词就是贪婪匹配
# 元字符后有一个量词，然后后面在继续接 ? 就是惰性匹配

# 贪婪匹配和惰性匹配例子
# 原字符串：玩吃鸡游戏，晚上一起上游戏，干啥呢？打游戏啊

玩.*游戏   贪婪匹配将最长匹配的结果返回
玩.*?游戏  惰性匹配将最短匹配的结果返回

~~~
