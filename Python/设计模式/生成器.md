
~~~python
# 生成器函数中包含 yield 关键字#，yield 关键字的作用和 return 关键字类似
# 在调用包含 yield 关键字的生成器函数时不会执行函数内部的代码，而是返回一个迭代器
# 使用调用生成器函数得到的迭代器进行一次迭代会执行生成器函数中的代码一直到 yield 关键字返回数据
# 而使用迭代器进行下一次迭代会接上上一次迭代yield 关键字返回数据的代码接着执行代码直到再次出现 yield 关键字返回数据
# 简而言之就是函数中的代码分段执行，并且分段返回数据

# 使用场景：当函数返回大量的数据时，内存不够使用，可以使用yield 关键字分段迭代调用代码返回数据

def order():
    lst = []
    for i in range(1, 10000000):
        lst.append(f"生产的第{i}件衣服")
        if len(lst) == 50 :
            yield lst
            lst=[]

# 调用生成器函数时不会执行函数内部的代码，而是返回一个迭代器
gen = order()

# 进行迭代，每次生产50件衣服返回
print(gen.__next__())
print(gen.__next__())
~~~
