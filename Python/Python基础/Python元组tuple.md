~~~python
# 元组是不允许修改元素的列表，元组的声明使用小括号而不是中括号
t = ("257", "241", "113", "132", "232",["qqwxe","qce3we"])

# 元组的元素修改会报错，但是元组元素内部的数据是可变的
t[1]="1231"
t[5][0]= "qsqw"
print(t)


# 如果元组只有一个元素时需要在声明时在元素的末尾加逗号，否则会认为小括号是运算符而不是元组声明
t1=("qdqwd",)

# 通过索引获取元组中的元素，如果是负数就是从后向前获取元素，但是无法超过元组的边界否则抛出异常
print(t[1])

# 元组的截取
# s[start:end]
# step表示步长，默认不写是1，正数代表从左向右截取元组，负数表示从右向左截取元组，并且数字表示每几个元素中取开头的元素
# step如果是正数那么start的索引必须在end索引左边，step如果是负数数那么start的索引必须在end索引右边
# 截取的元组包含start索引位置的元素但是不包含end索引位置元素
# 如果start省略就是截取元组开头到end索引位置元素的元组
# 如果end省略就是截取start索引位置到元组的结尾的子元组
# 如果是 s[:] 就还是原来的元组
print(t[2:5])

~~~
